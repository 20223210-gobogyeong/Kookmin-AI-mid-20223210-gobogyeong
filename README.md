# Arch-Sync | 개인용 프로젝트 관리 대시보드

## 📌 개요

Arch-Sync는 개인이 설계 스튜디오 프로젝트를 진행할 때, 자신의 할 일, 중요 일정, 리서치 링크를 한곳에 모아 체계적으로 관리하고 진행 상황을 놓치지 않도록 돕는 개인용 웹 대시보드입니다.

### 프로젝트 목표
- 개인 프로젝트 작업의 할 일, 일정, 리소스를 통합 관리
- 중요 일정과 마감일을 한눈에 확인
- 프로젝트 관련 링크와 리소스를 카테고리별로 아카이브
- 개인 메모 및 피드백을 체계적으로 기록

## ⚙️ 주요 기능

### 1. 대시보드
- **오늘 내가 할 일**: 마감일이 오늘인 할 일 목록 표시
- **최근 내 메모/피드백**: 최근에 작성한 메모/피드백/회의록을 최신순으로 표시
- **일정 달력**: 월별 달력으로 일정을 시각적으로 확인 (디데이 표시 포함)
- **다가오는 중요 일정**: 향후 일정을 날짜순으로 표시 (D-0, D-1, D-N 형태)

### 2. 할 일 관리 (칸반 보드)
- 드래그 앤 드롭으로 할 일 상태 변경 (대기 → 진행 중 → 완료)
- 할 일 추가/수정/삭제
- 담당자 정보 및 마감일 설정
- 상태별 칼럼으로 작업 진행 상황 시각화

### 3. 리소스 아카이브
- 프로젝트 관련 링크 저장 (구글 드라이브, 핀터레스트, 레퍼런스 사이트 등)
- 카테고리별 관리 (도면 링크, 패널 링크, 리서치 사이트)
- 카테고리별 필터링 기능
- 버전 정보 및 등록자 메모 기능

### 4. 공지 및 피드백 노트
- 개인 메모 저장 (피드백, 회의록, 아이디어)
- 유형별 필터링 (피드백, 회의록, 아이디어)
- 제목과 내용으로 상세 기록
- 작성자 정보 메모 기능

### 5. 일정 관리
- 중요 일정 등록 및 관리
- 디데이 자동 계산 (D-0: 오늘, D-1: 내일, D-N: N일 후)
- 긴급 일정 강조 표시 (D-3 이내 일정)
- 날짜순 정렬 및 리스트 뷰

## 🛠 기술 스택

- **프론트엔드**: HTML5, CSS3, JavaScript (Vanilla)
- **데이터 저장**: localStorage (브라우저 로컬 저장소)
- **외부 라이브러리**:
  - Google Fonts (Orbitron, Exo 2)
  - Font Awesome 6.4.0 (아이콘)
- **스타일링**: 순수 CSS (CSS Variables, Grid, Flexbox)
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 지원

## 🚀 실행 방법

1. 프로젝트 파일을 로컬에 다운로드
2. `index.html` 파일을 웹 브라우저로 직접 열기
   - 또는 로컬 웹 서버에서 실행 (예: `python -m http.server`, `npx serve`)
3. 브라우저에서 자동으로 localStorage가 활성화되어 데이터가 저장됨

### 최초 실행 시
- 샘플 데이터가 자동으로 초기화됩니다
- 이후 모든 데이터는 브라우저의 localStorage에 저장되어 페이지를 새로고침해도 유지됩니다

### 브라우저 호환성
- Chrome, Edge, Firefox, Safari 등 최신 브라우저에서 정상 작동
- localStorage를 지원하는 모든 브라우저에서 사용 가능

## 📝 주요 특징

- **오프라인 동작**: 인터넷 연결 없이도 작동 (초기 로딩 후)
- **데이터 영속성**: localStorage를 사용하여 브라우저에 데이터 저장
- **반응형 UI**: 모바일, 태블릿, 데스크톱 모두 지원
- **직관적인 UX**: 드래그 앤 드롭, 모달, 필터링 등 편리한 인터페이스
- **디데이 자동 계산**: 일정의 남은 일수를 자동으로 계산하여 표시

## ⚠️ 기타 참고사항

### 데이터 저장 위치
- 모든 데이터는 브라우저의 **localStorage**에 저장됩니다
- 브라우저의 쿠키/캐시 삭제 시 데이터가 삭제될 수 있습니다
- 다른 브라우저나 다른 컴퓨터에서는 데이터를 공유할 수 없습니다 (개인용 도구)

### 브라우저 제한 사항
- **Same-Origin Policy**: 다른 도메인에서 실행 시 localStorage 접근이 제한될 수 있습니다
- **시크릿 모드**: 일부 브라우저의 시크릿 모드에서는 localStorage가 비활성화될 수 있습니다

### 주요 기능 사용 팁
- **할 일 추가**: "할 일" 탭에서 "할 일 추가" 버튼 클릭
- **일정 추가**: "일정" 탭에서 "일정 추가" 버튼 클릭
- **칸반 보드**: 할 일 카드를 드래그하여 다른 상태로 이동
- **필터링**: 리소스와 피드백 탭에서 카테고리/유형 버튼 클릭
